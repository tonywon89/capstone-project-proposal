# API Endpoints

# HTML API

### Root

- `GET /` - loads React web app

### Users

- `GET /users/new`
- `POST /users`
- `PATCH /users`

### Session

- `GET /session/new`
- `POST /session`
- `DELETE /session`

## JSON API

### Feeds
- `GET /api/feeds`

- `GET /api/feeds/:id`

### Collections
- `GET /api/users/:user_id/collection`

- `POST /api/collections`

- `GET /api/collections/:id`

- `PATCH /api/collections/:id`

- `DELETE /api/collections/:id`

### Articles
- `POST /api/articles/:id`

- `DELETE /api/articles/:id`
