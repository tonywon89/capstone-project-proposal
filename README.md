#FeedMeScience

## Minimum Viable Product

FeedMeScience is a web application inspired by Feedly that will be built using Ruby on Rails and React.js. By the end of Week 9. this app will, at a minimum, satisfy the following criteria:

- [ ] New account creation, login, and guest/demo login
- [ ] Smooth, bug-free navigation
- [ ] Adequate seed data to demonstrate the site's features
- [ ] The minimally necessary features for a Feedly-inspired site: creating collections, adding or removing feeds to his collections, displaying the feeds of the user, and providing feeds for the user to choose from
- [ ] Hosting on Heroku
- [ ] CSS styling that is satisfactorily visually appealing
- [ ] A production README, replacing this README
